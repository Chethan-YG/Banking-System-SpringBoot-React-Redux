## ğŸš€ **Installation**

### **Backend (Spring Boot)**
1. Navigate to the backend folder:
   ```sh
   cd bank-backend
   ```
2. Configure `application.properties` with your database credentials.
3. Build and run:
   ```sh
   mvn clean install
   mvn spring-boot:run
   ```

### **Frontend (React)**
1. Navigate to the frontend folder:
   ```sh
   cd bank-frontend
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm start
   ```
